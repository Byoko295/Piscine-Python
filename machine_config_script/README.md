# Group 1072517

Welcome to the repository for Group 1072517! This repository contains various scripts and tools developed as part of our collaborative project. Below, you'll find an overview of the repository structure and instructions for getting started.

## Repository Structure

- **git_setters/**: Scripts for configuring Git settings.
- **validators/**: Validation scripts for various system configurations, including Git, text editors, and package installations.
- **vim_setters/**: Scripts for setting up Vim configurations.
- **setup.sh**: A setup script to initialize the environment.

## How to Use

1. Clone the repository:

   ```bash
   git clone https://rendu-git.etna-alternance.net/module-10279/activity-55056/group-1072517.git
   ```

2. Navigate to the repository directory:

   ```bash
   cd group-1072517
   ```

3. Run the setup script to initialize the environment:

   ```bash
   sudo ./setup.sh
   ```

4. Explore the subdirectories for specific tools and scripts.
